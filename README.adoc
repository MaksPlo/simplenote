# SimpleNote

=====Introduction

SimpleNote is a demo project wich use technologies and tools on backend:

- Java 8
- Spring boot
- Spring Security
- Spring Data
- Junit
- Mockito
- FlyWay
- PostgreSQL
- Mongo DB
- Swagger
- Lombok

and on frontend:

- JS
- Vue
- Vuetify
- Node.js

==How it work==

Project start from login form. 
For authentication, gmail OAuth 2.0 is used, the token is provided by Google API.
